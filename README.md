<h2>Academic Event Manager</h2>
<p>Welcome to the Academic Event Manager, a comprehensive web-based platform designed to streamline the planning and management of college events in academic institutions.</p>
<h2>Overview:</h2>
<p>The Academic Event Manager offers a structured workflow for event planning, proposal submission, and approval, facilitating collaboration among faculty, department heads (HODs), and the principal. The platform aims to enhance efficiency, communication, and decision-making in organizing academic events.</p>
<h2>Key Features:</h2>
<ul>
  <li><b>Role-Based Access : </b> Users are assigned specific roles (faculty, HOD, principal) with distinct permissions to ensure appropriate access and responsibilities.</li>
  <li><b>Event Proposal Workflow : </b>  Faculty can submit event proposals, requirements, and budgets for review and approval by HODs and the principal.</li>
  <li><b>User-Friendly Interface : </b>  Intuitive navigation and visually appealing design enhance user experience.
</li>
  <li><b>Database Management :</b>MySQL database ensures secure storage and retrieval of event-related information.
</li>
</ul>

<h2>Technologies Used :</h2>
<ul>
  <li>Frontend: HTML, CSS, JavaScript, uiverse.io for animations and visual effects</li>
  <li>Backend: PHP</li>
  <li>Database: MySQL</li>
</ul>
